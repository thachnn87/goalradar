# Session Start Checklist

Status: Current
Owner: Project maintainer
Last Reviewed: 2026-06-29
Update Trigger: Update when the required agent start/finish workflow changes.
Authority: Operational checklist. Lower authority than `.ai/AI_RULES.md`.

Before writing code:

1. Read `docs/PROJECT_CONTEXT.md`
2. Read `.ai/AI_RULES.md`
3. Read `.ai/CURRENT_SPRINT.md`
4. Read `.ai/HANDOFF.md` when continuing or handing off work
5. Check `git status --short`
6. Confirm the user request is code, docs, audit, or planning work

After finishing:

1. Update `.ai/CURRENT_SPRINT.md` only when sprint state changes
2. Append `.ai/CHANGELOG.md` only for material AI-assisted repository changes
3. Update `.ai/HANDOFF.md` when handing off or leaving incomplete work
4. Do not modify `docs/PROJECT_CONTEXT.md` unless stable project context changes
5. Summarize files changed and any manual follow-up
