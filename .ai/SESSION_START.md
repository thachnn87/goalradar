# Session Start Checklist

Before writing code:

1. Read `docs/PROJECT_CONTEXT.md`
2. Read `.ai/AI_RULES.md`
3. Read `.ai/CURRENT_SPRINT.md`
4. Read `.ai/HANDOFF.md`
5. Check `git status --short`
6. Confirm the user request is code, docs, audit, or planning work

After finishing:

1. Update `.ai/CURRENT_SPRINT.md`
2. Append `.ai/CHANGELOG.md`
3. Update `.ai/HANDOFF.md`
4. Do not modify `docs/PROJECT_CONTEXT.md` unless architecture or project state changes
5. Summarize files changed and any manual follow-up
